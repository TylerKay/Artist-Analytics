Running the program:
	1. Make sure you have node.js installed.
	2. In the project's directory, run npm install -g http-server
	4. Run http-server -c-1
	5. On your browser, go to http://127.0.0.1:8080
	3. Type in the artist name and the filter you would like to apply. You can view the artist names in the dataset (artists.csv)
	3. Click submit and your bar graph will display

Dataset used:
- https://www.kaggle.com/datasets/saurabhshahane/music-dataset-1950-to-2019?resource=download This dataset uses the Attribution 4.0 International (CC BY 4.0) license
